This Github repository holds our project, which has the following:

  • File In/Out: New, Open (txt,odt,pdf to text), Save As (txt,odt,pdf), Search (1 to many), Exit
  • SCPC Features: Select, Cut, Copy, Paste, Add Timestamp
  • About: You are currently reading it!
  • Print to system printer
  • Github actions CI is implemented, with Maven building and testing on integration into the main branch
  • Testing implemented using TestFX framework, and as required it's FxRobot to mimic a user and test GUI elements
  • YaML based configuration
